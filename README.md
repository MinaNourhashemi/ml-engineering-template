# ML Engineering Template

A minimal machine learning engineering template with:

- Ruff for linting and formatting
- pre-commit hooks
- pytest unit tests
- basic structured logging

## Project Structure

- `src/app/` → application code
- `tests/` → unit tests
- `scripts/` → runnable scripts

## Commands

pytest  
ruff check .  
ruff format .  
python scripts/run_training.py# ml-engineering-template
